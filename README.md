# 🚀 Tech Evolution

Projeto final do curso **"Página web: do zero à primeira interação"** - Start by Alura.

## Sobre o projeto

Uma rede social fictícia focada em **tecnologia e evolução pessoal**.  
O site contém um feed de posts com:

- HTML5 semântico
- CSS com Flexbox e design moderno (gradiente dark tech)
- JavaScript interativo (sistema de likes com contador em múltiplos posts)

## Funcionalidades

- Design responsivo
- 7 posts sobre motivação tech e desenvolvimento
- Botão de like com contagem (curtir e descurtir)
- Efeito de animação ao curtir
- Hover effects modernos

## Tecnologias utilizadas

- **HTML5**
- **CSS3** (Flexbox + Gradientes)
- **JavaScript** (DOM manipulation)

## Como rodar localmente

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Clique nos ❤️ para testar os likes!
